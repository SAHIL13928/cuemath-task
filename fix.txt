File: app/deck/[id]/page.tsx (or wherever DeckPage component lives)

before any if/return statements.

STEP 5 — Fix the multiple GoTrueClient instances warning.
Find all places where createClient() or createBrowserClient() is called
outside of a component or hook (i.e. at module level):

grep -r "createBrowserClient\|createClient" --include="*.ts" --include="*.tsx" lib/ components/

Any file that calls createBrowserClient at the TOP LEVEL (not inside a function)
needs to be wrapped in a singleton pattern:

import { createBrowserClient } from '@supabase/ssr'

let client: ReturnType<typeof createBrowserClient> | null = null

export function getSupabaseBrowserClient() {
  if (!client) {
    client = createBrowserClient(
      process.env.NEXT_PUBLIC_SUPABASE_URL!,
      process.env.NEXT_PUBLIC_SUPABASE_ANON_KEY!
    )
  }
  return client
}

Replace all direct createBrowserClient() calls in client components
with getSupabaseBrowserClient() from this singleton.
